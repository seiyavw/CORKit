#CORKit

This framework is currently in alpha.

### Extensions
- CORPagingScrollView
- CORScrollMenuBar
- CORAudioSlider
- CORDefaultCell
- CORBaseViewController
- CORBlendView
- CORNavigationButton
- CORWindowView

- NSDate

- UIImage
- UIView
- UIViewController
- UIStoryboard
